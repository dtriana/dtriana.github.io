Skype 4.1 with desktop sharing

August 5, 2009 1-minute read
Uncategorized

Bueno, no exáctamente desktop sharing como lo conocemos a traves de herramientas como remote desktop o sharedview pero si permite brindarle al interlocutor una especie de pantalla compartida.

Ventajas frente a Microsoft Sharedview o Microsoft Office Live Meeting

  • No se necesita otro producto aparte para el audio y otro para la presentación
  • No degrada el video a Vista Basic como si ocurre con los productos de Microsoft al ejecutarlos en Vista con Aero
  • Permite compartir la otra pantalla en el caso de configuraciones Dual Screen (Cosa que no hacen los productos de Microsoft)

Limitaciones

  • No permite ceder el control
  • No permite tomar notas en pantalla
  • No permite resaltar ni señalar nada en la pantalla

Resulta una buena alternativa para demostraciones a clientes, charlas, capacitaciones y soporte remoto aunque claramente sus limitaciones pueden hacerlo poco útil en muchos escenarios, aunque seguramente es solo cuestión de un par de nuevas versiones para skype quede a la par de las demas soluciones.
