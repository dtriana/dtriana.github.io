Teracopy

September 17, 2008
Uncategorized

Hace un par de podcasts Paul Thurrott de Windows Weekly recomendó una utilidad llamada Teracopy como algo indispensable para reemplazar el abusurdamente lento proceso de copia en Windows Vista.

Tuve que hacer espacio para instalar un dual boot con Windows XP en el Tablet para poder utilizar un cable USB - Serial que requiero para configurar unos equipos de un cliente. Como necesitaba tanto espacio tuve que borrar las imágenes de las máquinas virtuales (40Gb) que utilizo tanto el el desktop como en el tablet. Cuando las copié por primera vez Vista tardó casi 4 horas copiandolas via ethernet a 100mbps. La velocidad que reportaba Vista a veces subia hasta 10MB, pero regularmente estaba en 6.5MB y podia bajar hasta 4MB asi que que mejor momento para probar la recomendación de Paul.

Instalé el Teracopy; ningun problema, instalador liviano, sin crapware, …

Inicié la copia y tará!!!, 11MB de transferencia sostenidos, 1 hora en total copiando, como quien dice, MUST HAVE, muy recomendado.

Se baja de aqui.
